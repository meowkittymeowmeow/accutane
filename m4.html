module 1 — permutations, combinations, counting, equally-likely events

1. equally likely outcomes

if a finite sample space S has N simple outcomes and each has probability 1/N, then:

P(A) = |A| / N

example: roll a fair die twice → 36 ordered pairs.

  • event “first roll is 1”: 6 outcomes → 6/36 = 1/6
  • event “sum is 8”: 5 outcomes → 5/36
  • event “second roll = first roll + 2”: 4 outcomes → 4/36 = 1/9

2. permutations (order matters)

choosing ordered k-tuples from n distinct objects:

P_{k,n} = n (n−1)(n−2)…(n−k+1) = n! / (n−k)!

example: president / VP / treasurer from 60 people:

P_{3,60} = 60·59·58 = 60!/57! = 205,320

3. combinations (order does not matter)

unordered selection of size k from n:

C(n, k) = n! / (k! (n−k)!)

motivation: each unordered group corresponds to k! permutations, so dividing collapses reorderings.

example: choose any 3-person team from 60:

C(60, 3) = 60! / (3!·57!)

symmetry: C(n, k) = C(n, n−k)

4. committee example (mixed groups)

60 people: 35 women, 25 men. select an 11-person committee.

sample space size:

C(60, 11)

probability committee has at least 5 men AND at least 5 women:

only two valid splits: (5 men, 6 women) and (6 men, 5 women).

P = [ C(25,5) C(35,6) / C(60,11) ] + [ C(25,6) C(35,5) / C(60,11) ]

5. bus-inspection example

20 buses → 8 cracked, 12 uncracked. sample 5 at random.

sample space:

C(20, 5)

probability exactly 4 have cracks

[ C(8,4) C(12,1) ] / C(20,5)

probability at least 4 have cracks

[ C(8,4) C(12,1) ] / C(20,5) + [ C(8,5) C(12,0) ] / C(20,5)
