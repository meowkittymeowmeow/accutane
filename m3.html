module 1 — axioms + basic probability

1. goal

  • assign a number P(A) to an event A that measures its chance of occurring.
  • probability theory: assumes the population model is known → computes event probabilities.
  • statistics: observes a sample → estimates those probabilities.

2. setup recap

  • experiment → produces outcomes.
  • sample space S → all possible outcomes.
  • event A ⊆ S.
  • we want a consistent definition of P(A).

3. the three axioms

  1. non-negativity + boundedness: 0 ≤ P(A) ≤ 1 for any event A.
  2. certainty of the sample space: P(S) = 1.
  3. additivity for mutually exclusive events: if A₁, …, Aₙ are pairwise disjoint, then
    P(A₁ ∪ ··· ∪ Aₙ) = P(A₁) + ··· + P(Aₙ).
    same idea extends to a countably infinite disjoint collection.

4. example — flip a fair coin until first tail

sample space: {T, HT, HHT, HHHT, …}

let Aₙ be “tail occurs on the n-th flip”.

then P(Aₙ) = 1 / 2ⁿ, and the Aₙ are mutually exclusive.

check the axioms: the events cover the whole space and

P(S) = Σₙ₌₁^∞ (1 / 2ⁿ) = 1.

event B: tail occurs after at least 3 flips

B = sequences starting with at least two H before the first T.

easier via the complement. Bᶜ = get a tail on flip 1 or 2:

  • tail on flip 1: P = 1/2
  • head then tail: P = (1/2)·(1/2) = 1/4

disjoint, so

P(Bᶜ) = 1/2 + 1/4 = 3/4

and

P(B) = 1 − P(Bᶜ) = 1/4.

5. consequences of the axioms

  • A ∪ Aᶜ = S and A ∩ Aᶜ = ∅ ⇒ P(Aᶜ) = 1 − P(A).
  • if A ∩ B = ∅ then P(A ∩ B) = 0.
  • general union formula:
    P(A ∪ B) = P(A) + P(B) − P(A ∩ B).

6. car-defect example

three defect types → events A, B, C. given:

event P(·)
A0.20
B0.25
C0.30
A ∩ B0.05
B ∩ C0.075
A ∩ C0.06
A ∩ B ∩ C0.015

(a) probability defect 1 does not occur

P(Aᶜ) = 1 − P(A) = 1 − 0.2 = 0.8.

(b) probability at least one defect

use inclusion–exclusion for three sets:

P(A ∪ B ∪ C) = P(A) + P(B) + P(C)
− P(A ∩ B) − P(B ∩ C) − P(A ∩ C)
+ P(A ∩ B ∩ C)

plug numbers:

0.2 + 0.25 + 0.3 − 0.05 − 0.075 − 0.06 + 0.015 = 0.7.

(c) probability of no defect

this is (A ∪ B ∪ C)ᶜ, so

P(none) = 1 − 0.7 = 0.3.

(d) probability defects 1 and 3 occur, but not 2

this is the event 101. take A ∩ C and remove cases where B also occurs:

P(101) = P(A ∩ C) − P(A ∩ B ∩ C)
= 0.06 − 0.015 = 0.045.
